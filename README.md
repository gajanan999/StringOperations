# StringOperations

### Longest String API :
    Method : POST
    URL : http://localhost:8080/getLongestString
    Body:
    {
        "list":[
            "adf",
            "sadas",
            "sadfasfadf",
            "dsf",
            "sfsdfjsdjfgklsdjgkljsl",
            "d"
        ]
    }

### Delete Strings API

    Method: POST
    URL: http://localhost:8080/deleteStrings?length=10
